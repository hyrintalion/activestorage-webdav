# ActivestorageWebDAV

Webdav service for activestorage.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'activestorage_webdav'
```

And then execute:

    $ bundle

Set up upyun storage service in config/storage.yml:

```yml
webdav:
  service: WebDAV
  url: "http://path_to_your/webdav_server/"
```
